# 魔音 v1.0.4 发布

## 修复内容
- 修复 Android 音效功能（PresetReverb 预设值修正）
- 包名从 `com.my.music.moblie` 更改为 `cn.my.music.mobile`
- 修复 Gradle 构建配置

## APK 下载

| 架构 | 文件 | 大小 |
|------|------|------|
| Universal (推荐) | my-music-v1.0.4-universal.apk | 66MB |
| arm64-v8a | my-music-v1.0.4-arm64-v8a.apk | 27MB |
| armeabi-v7a | my-music-v1.0.4-armeabi-v7a.apk | 25MB |
| x86 | my-music-v1.0.4-x86.apk | 29MB |
| x86_64 | my-music-v1.0.4-x86_64.apk | 29MB |

## 安装说明
1. 下载对应架构的 APK 文件
2. 在手机上启用"安装未知来源应用"权限
3. 点击 APK 文件进行安装
